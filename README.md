## 게임 소개

장르 : 3D FPS, 슈팅게임

플랫폼 : PC (Windows)

제작 기간 : 2개월(2023. 09. 04. ~ 2023. 10. 25.)

사용 도구 : Unity 2022.3.7.f1, VS 2022, Github

개발 인원 : 3명(개발자 2명)

게임 소개 : 화염총으로 쿠키와 마녀를 물리치고 그레텔을 구하는 게임

## 게임 플레이

### 조작법

WASD : 이동

마우스 좌클릭 : 발사

E : 냄새 스킬

## 팀원 소개

팀장 - 전신영

개발 - 장명훈, 전신영

디자인 - 신주홍
